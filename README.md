## 格式建议

1. 区分好 “的” “地” “得”；
2. 英文单词的首位要加一个空格，主要是单词出现在中文内容时；
3. 除了引用的英文外，文中不要出现英文标点；
4. 中文的出现英文名词复数格式改为单数；
5. 英文出现在句首大写；
6. 英文中貌似没有“、”，按照语义将英文逗号改为中文顿号；
7. 避免文中出现错别字；
